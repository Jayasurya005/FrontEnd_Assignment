<!---- EXPLANATION ---- !>
1. run the index.html file
2. index.html contains a search bar and a button .
3. on clicking the search button after entering country name
   it makes an API call using fetch API
4. Then it displays the flag and country name as a card
5. on clicking the card  it redirects to the country.html file
   where the country's description are shown 
6. country.html has a back button to go back to the index.html page again.
